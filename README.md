Store
=====

Lightweight configuration storage.

Requires [node](http://nodejs.org) and [npm](http://www.npmjs.org).

## Install

```
npm i cli-store
```

## Developer

### Test

```
npm test
```

### Documentation

To generate all documentation:

```
npm run docs
```
